安装 Flash 插件（不推荐）

安装 Flash 插件（不推荐）

注意：该插件下载安装在国内非常缓慢，甚至会导致更新卡死！

Chrome 已经自带了 Flash 插件，该插件仅供 Firefox 和 Chromium 使用，在 Flash 逐渐淘汰的今天，我们完全可以选择不安装它。

sudo apt update
sudo apt install flashplugin-installer
