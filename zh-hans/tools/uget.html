安装 uGet

安装 uGet

uGet 是一款 Linux 下支持多点下载和断线续传的下载工具。

sudo apt update
sudo apt install aria2 uget

安装完成以后请在 uGet 的设置选项中的Plugin面板开启aria2支持，这样能够在以后下载时使用多点下载功能。
